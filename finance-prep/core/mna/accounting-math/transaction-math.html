2.4 Transaction Math

Mülakatlarda kaðýt kalemle çözdürülen o meþhur matematik.

Offer Value to Enterprise Value

1. Offer Price: Hisse baþý teklif (Örn: 20$).
2. Equity Value: 20$ * Seyreltilmiþ Hisse Sayýsý = 200M $.
3. Refinancing: Hedefin mevcut borcunu (50M $) ödemek zorundasýnýz.
4. Transaction Value: 200M (Hisse) + 50M (Borç ödemesi) = 250M $. (Ýþlemin toplam maliyeti).

Pitfall: Diluted Shares’ý Unutma

Offer price * basic shares yapmak hata. Opsiyon/RSU’lar (TSM) diluted share sayýsýný artýrýr; equity value ve transaction value deðiþir.

Rapid Fire Drills

Soru: Neden “cash” EV bridge’de çýkarýlýr?

Çünkü þirketi satýn aldýðýnda kasadaki nakit de sana geçer; net maliyeti düþürür (excess cash mantýðý).

Soru: Transaction value ile equity value ayný þey mi?

Hayýr. Equity value sadece hissedara ödenen kýsým. Transaction/enterprise value ise borç gibi diðer paydaþlarýn da dahil olduðu “þirketin tamamýna” ödenen bedeldir.
