2.4 Transaction Math

Mülakatlarda kağıt kalemle çözdürülen o meşhur matematik.

Offer Value to Enterprise Value

1. Offer Price: Hisse başı teklif (Örn: 20$).
2. Equity Value: 20$ * Seyreltilmiş Hisse Sayısı = 200M $.
3. Refinancing: Hedefin mevcut borcunu (50M $) ödemek zorundasınız.
4. Transaction Value: 200M (Hisse) + 50M (Borç ödemesi) = 250M $. (İşlemin toplam maliyeti).

Sources & Uses

Parayı nereden bulduk (Sources), nereye harcadık (Uses)? Her zaman eşit olmalıdır.

Sources:
+ New Debt (Bankadan Kredi): 100M
+ New Equity (Yatırımcıdan Para): 150M
+ Cash on Hand (Kasadaki Para): 10M
= Total Sources: 260M

Uses:
- Purchase of Equity (Hissedarlara Ödeme): 200M
- Refinance Debt (Eski Borcu Kapama): 50M
- Transaction Fees (Masraflar): 10M
= Total Uses: 260M
