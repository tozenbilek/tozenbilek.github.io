Kural 2: Fare Kullanmak Yasak (Mouse-Free)

Hızlı olmak zorundasınız. Fare kullanmak sizi yavaşlatır. İşte hayat kurtaran kısayollar:

  • Alt + E + S + V : Paste Special Values (Değerleri yapıştır).
  • Alt + E + S + T : Paste Formats (Formatı yapıştır).
  • Ctrl + [ : Formülün kaynağına git (Trace Precedents).
  • F5 + Enter : Geri dön (Go To).
  • Alt + = : AutoSum (Otomatik Topla).

Kural 3: "Hardcode"ları Ayrıştır

Asla formülün içine elle sayı girmeyin.
YANLIŞ: = A1 * 1.05 (O %5 nereden geldi? Kimse bilemez).
DOĞRU: = A1 * (1 + B1) (B1 hücresinde %5 growth rate açıkça yazmalı).
