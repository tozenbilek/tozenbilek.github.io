8.2 Fulcrum Security

Restructuring mülakatlarının en önemli kavramıdır. "Şirketin değeri bittiği yerde, anahtarlar kime teslim edilir?"

Waterfall Logic

Varlık Değeri: 100M $.

Borç Sıralaması:
1. Secured Bank Debt (50M): Tamamını alır (100% Recovery). (Kalan Varlık: 50M).
2. Senior Notes (40M): Tamamını alır (100% Recovery). (Kalan Varlık: 10M).
3. Subordinated Notes (50M): Sadece 10M kalmıştır. (20% Recovery).
-> İşte Fulcrum Security budur!
4. Equity (Hissedarlar): Hiçbir şey alamaz (Wiped out).

Sonuç

Fulcrum Security sahipleri (Örneğimizde Subordinated Notes), şirketin yeni sahipleri (New Equity) olurlar. Paralarının tamamını alamadıkları için, borçları silinir ve karşılığında şirketin hisselerini alırlar.
