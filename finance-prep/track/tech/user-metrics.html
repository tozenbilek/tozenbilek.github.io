3.3 User Metrics

Henüz para kazanmayan (Pre-revenue) uygulamaları nasıl değerleriz? Kullanıcılarına bakarak.

Volume (Hacim)

  • DAU (Daily Active Users): Günlük aktif kullanıcı. (Bağımlılık göstergesi).
  • MAU (Monthly Active Users): Aylık aktif kullanıcı. (Erişim göstergesi).

Engagement (Bağlılık)

DAU / MAU Ratio: Bir kullanıcı ayda kaç gün giriyor?
Örn: Facebook %66 (Çok yüksek). LinkedIn %40.

Monetization (Paraya Çevirme)

ARPU (Average Revenue Per User): Kullanıcı başına ortalama gelir.
Yatırımcılar şuna bakar: "Kullanıcı tabanı var, henüz ARPU düşük. İleride reklam göstererek ARPU'yu artırabilirler mi?" (Potansiyel).
