# System Programming

Bu bölümde, sistem programlamanın temel taşlarını ele alan notlar bulunmaktadır. Konular, C programlama dilinin temellerinden başlayarak bit ve byte'ların dünyasına, assembly diline, bellek hiyerarşisine ve derleme süreçlerine kadar uzanmaktadır.
